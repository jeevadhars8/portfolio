My Projects

Some Of My Distinguished Works

This project utilizes radar-based micro-Doppler signals to classify aerial targets like drones and birds. By applying Short-Time Fourier Transform (STFT), signals are converted into spectrograms for analysis. Machine learning models, developed using TensorFlow/PyTorch, enhance classification accuracy. Background noise is filtered using matched filtering to maximize the signal-to-noise ratio. The system improves surveillance, security, and autonomous navigation applications. Scalable and efficient, it supports integration into modern radar and multi-sensor fusion systems. Let me know if youd like any modifications!

Micro-Doppler Based Target Classification Using Machine Learning

This project focuses on developing a non-electrical solar tracking system using Shape Memory Alloy (SMA) to enhance solar panel efficiency. The system leverages Nitinol SMA wires, which contract when heated by sunlight, automatically adjusting the panel's tilt for optimal alignment. A hinged mounting frame and return spring mechanism ensure smooth movement and reset. The self-powered, cost-effective, and low-maintenance design makes it ideal for off-grid and remote locations. This innovation improves solar power generation by up to 30% without requiring external power sources.

Development of a Non-Electrical Solar Tracker Using Shape Memory Alloys (SMA)
